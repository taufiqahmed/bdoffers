#BDOffers in ReactJs

Trying to learn ReactJS with a real life project called BDoffers, an offer listing webpage.

##Features:

- Show a list of Offers by
  - Brand Name
  - Discount
  - Title
  - Expiry Date
- Single Page for Showing details of the offer including
  - Brand Outlets
  - Brand Contact Info
- Brand Page
  - Address of Brand Outlets
  - Past Offers
